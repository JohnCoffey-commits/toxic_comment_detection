# Data Dictionary

| Column | Description |
| --- | --- |
| `sample_id` | Deterministic row identifier generated by source and retained row order. |
| `source_dataset` | `jigsaw` or `civil_comments`. |
| `raw_text` | Closest available original comment text; not lowercased. |
| `text_clean` | Lightly normalized text for transformer models such as DistilBERT. |
| `text_tfidf` | Lowercased lexical view for TF-IDF + Logistic Regression; punctuation retained. |
| `clean_text` | Simplified teammate-facing CSV text column derived from `text_clean`. |
| `binary_label` | Unified toxic vs non-toxic label where 1 means toxic. |
| `label` | Simplified teammate-facing CSV target column derived from `binary_label`. |
| `orig_label_info` | Compact JSON string preserving source labels or scores. |
| `char_len` | Character length of `text_clean`. |
| `word_len` | Whitespace token count of `text_clean`. |
| `text_hash_raw` | SHA-256 hash of `raw_text`. |
| `text_hash_normalized` | SHA-256 hash of duplicate-normalized text. |
| `bert_token_len` | DistilBERT tokenizer length including special tokens. |
| `split` | Split assignment. |
| `has_identity_term` | Reusable identity challenge-slice flag. |
| `has_obfuscation` | Reusable obfuscation challenge-slice flag. |
| `implicit_proxy` | Proxy flag for toxic rows without explicit lexicon terms; not a gold label. |
| `length_bucket` | `short`, `medium`, or `long` from saved thresholds. |
